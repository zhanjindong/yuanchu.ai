# 🌌 元初AI - YuanChu AI

> 宇宙、文明与万物的初始 · 一切智能的本源与开端

## 关于

元初AI是一个致力于通用人工智能研究与应用的创新平台。我们相信，AI应该服务于人类，成为推动文明进步的力量。

"元初"取自中国传统文化，象征初始、本源与开端。我们致力于构建更安全、更强大、更普惠的人工智能系统。

## 快速开始

### 本地开发

```bash
# 克隆仓库
git clone https://github.com/zhanjindong/yuanchu.ai.git
cd yuanchu.ai

# 本地预览
# 直接用浏览器打开 index.html
# 或使用本地服务器
python3 -m http.server 8000
```

访问 **[yuanchu.ai](https://yuanchu.ai/)**
